# 📝 To-Do Web Application

A responsive and modern **To-Do Web Application** developed using **HTML, CSS, and JavaScript** as part of **Task 04** of my **Web Development Internship at SkillCraft Technology**.

## 📌 Project Overview

This application helps users organize and manage daily tasks efficiently. It allows users to create, edit, delete, search, filter, and track tasks while storing data locally using the browser's Local Storage.

---

## 🚀 Features

- ✅ Add New Tasks
- ✏️ Edit Existing Tasks
- 🗑️ Delete Tasks
- ✔️ Mark Tasks as Complete/Incomplete
- 🔍 Live Search Functionality
- 🎯 Filter Tasks (All, Pending, Completed, High Priority)
- 📊 Dashboard Statistics
- 🌙 Dark & Light Mode
- 💾 Local Storage Support
- 🧹 Clear Completed Tasks
- 🗑️ Delete All Tasks
- 📱 Fully Responsive Design
- 🎨 Modern User Interface

---

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension.
4. Right-click on **index.html**.
5. Select **Open with Live Server**.
6. Start managing your tasks.
